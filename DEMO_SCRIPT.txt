[1:00 - ARCHITECTURE]
(show assets/architecture_diagram.png) (pause)
Let me walk you through our multi-agent pipeline node by node. First, the Intake Agent processes raw, unstructured citizen texts, parsing key metrics. Next, the Privacy Agent redacts personal details, and the Verification Agent checks GPS coordinates to merge duplicates. The Triage Agent then calculates an urgency score from 1 to 10 based on demographics. From there, the Resource Matcher Agent queries our MCP servers to link nearby volunteers and inventory. Finally, the Coordination Agent compiles the findings, logs the audit log, and places the ticket on the dashboard for human coordinator approval.
